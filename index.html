LEGO Mobile

LEGO Mobile 是 LEGO 移动前端开发框架，提供简单、快速的 Web 开发体验。LEGO Mobile 是基于 zepto 和 seajs 的轻量级移动组件库，提供了一系列webapp的UI解决方案。支持 UC，QQ，Opera，Safari，Chrome 等国内主流手机浏览器。
